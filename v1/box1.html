Connecting a WebTV box to WebTV Redialed
There are two options to connect a WebTV box to WebTV Redialed. You can use a real landline, or simulate one with a Raspberry Pi. If you'd like to use a real landline, follow the guide for "Connecting via a Landline", if you'd like to use a Pi, follow the guide for "Connecting via DreamPi".
